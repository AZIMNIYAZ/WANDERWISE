# WANDERWISE

WanderWise is a travel itinerary planning application designed to help users plan efficient and well-structured trips across destinations in India. The application allows users to select a destination and generates an optimized travel plan with curated recommendations.

The platform displays must-visit places along with essential travel details such as best visiting time, estimated duration, entry fees, weekly off days, and Google ratings. This helps travelers make informed decisions without manually researching multiple sources.

WanderWise also includes a budget estimation module to calculate the overall trip cost and integrates the Google Maps API for route visualization and navigation, providing a seamless end-to-end travel planning experience.
